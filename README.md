# 🍔 Grill & Co. – Fast Food Landing Page

A modern, appetizing landing page for a burger restaurant. Built with **Vanilla HTML, CSS & JavaScript** – no frameworks, just clean code.

![Desktop View](screenshots/desktop.png)

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🍔 **Modern Design** | Warm, appetizing colors (red & orange) |
| 📱 **Fully Responsive** | Perfect on mobile, tablet, and desktop |
| 🧀 **Menu Section** | Products with descriptions and prices |
| ⭐ **Customer Reviews** | Social proof with star ratings |
| 🚀 **Call-to-Action** | "Order Now" button for conversions |
| 📍 **Location & Hours** | Footer with business info |
| 🎨 **Custom Logo** | Branding with Fredoka One font |

---
